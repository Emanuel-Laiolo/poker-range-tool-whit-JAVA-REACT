# Range Builder (Python)

Copied from your drop folder into the project so the repo contains a working tool to create/export ranges as JSON.

## Run
From repo root:

```bash
python3 tools/range_builder/builder.py
```

## What it does
- Lets you paint hands on a 13x13 grid.
- Supports multiple action layers (OPEN/CALL/3BET/etc) with weights.
- Exports a JSON strategy file that can be used as test data for the Java+React app.
